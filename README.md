# Pages Theme

This is a Jekyll theme named "Pages Theme". It is designed to provide a clean and simple layout for your Jekyll site.

## Installation

To install the theme, add the following line to your Gemfile:

```ruby
gem 'pages-theme'
```

Then, run the following command to install the gem:

```
bundle install
```

## Usage

To use the theme in your Jekyll site, add the following line to your `_config.yml` file:

```yaml
theme: pages-theme
```

## Directory Structure

- `_layouts`: Contains layout files for the theme.
- `_includes`: Contains reusable components like the head section.
- `_sass`: Contains Sass files for styling.
- `assets`: Contains CSS and JavaScript files for the theme.
- `pages`: Contains Markdown files for content.
